# Teams In Town

![Output](src/images/games_view.png)

Game In Town will display every major sports game within a metro area in the US/Canada within a specific date range.

## Sports and Leagues Onboarded

-   Baseball
    -   MLB
-   Basketball
    -   NBA
    -   NCAA Men's
    -   NCAA Women's
-   Football
    -   NCAA
    -   NFL
-   Hockey
    -   NHL
-   Soccer
    -   MLS
